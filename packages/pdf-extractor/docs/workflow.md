# 操作フロー

このツールは以下の流れで使用します。

## 1. PDFをアップロード

アプリに PDF ファイルを読み込みます。

対象例

-   納品書
-   現品票
-   出荷指示書

## 2. 項目の座標を登録

PDF画面を見ながら、抽出したい項目の座標を指定します。

例

-   品番
-   数量
-   納期
-   発行番号

登録情報は **layout_schema.json** に保存されます。

## 3. レイアウトを保存

一度設定したレイアウトはテンプレートとして保存されます。

同じ帳票形式の PDF に再利用できます。

## 4. データ抽出

登録された座標を元に、PDFからテキストを取得します。

処理

    PDF
    ↓
    座標抽出
    ↓
    テキスト取得
    ↓
    データ整形

## 5. CSV出力

抽出されたデータは CSV としてダウンロードできます。

例

    item_code,quantity,delivery_date
    ABC123,10,2026-03-15

## 6. 業務システムへ取り込み

CSVデータを

-   WMS
-   在庫管理
-   出荷管理

などのシステムに取り込みます。
