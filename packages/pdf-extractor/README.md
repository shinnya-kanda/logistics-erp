# PDF Layout Extractor

### GUIでPDFの項目位置を登録しデータを抽出するツール

**PDF Layout Extractor** は、PDF帳票のレイアウト（座標）を登録し、
指定した位置からデータを抽出して **CSV化するためのツール** です。

固定レイアウトの帳票からデータを取得する用途を想定しています。

対象例

-   納品書
-   現品票
-   出荷指示書
-   請求書
-   製造帳票

一度レイアウトを登録すれば、同じ形式のPDFを **自動でCSV化**できます。

------------------------------------------------------------------------

# 特徴

### 1. GUIで項目位置を登録

PDF上の座標を指定してデータ取得位置を登録します。

処理イメージ

PDF ↓ 項目位置登録 ↓ レイアウト保存 ↓ データ抽出

------------------------------------------------------------------------

### 2. CSVデータ生成

登録された項目から以下のような情報を抽出できます。

例

-   品番
-   数量
-   指定納入日
-   発行番号
-   品名
-   荷降場

出力形式

CSV

------------------------------------------------------------------------

### 3. 同一帳票の自動処理

一度レイアウトを登録すると、同じ帳票形式のPDFを 繰り返し処理できます。

例

日次処理

PDF納品書 ↓ 自動抽出 ↓ CSV出力 ↓ 業務システム取込
